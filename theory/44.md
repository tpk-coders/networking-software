##JavaScript. Переменные

Обычная переменная:
```javascript
var message;
message = 'Привет';
alert(message); // выведет окно с переменной
```
Константа — это переменная, которая никогда не меняется. Как правило, их называют большими буквами, через подчёркивание.
```javascript
var COLOR_BLUE = "#00F";
alert(COLOR_BLUE);
```
