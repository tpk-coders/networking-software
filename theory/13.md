##HTML. Ссылки

```html
<!-- классическая ссылка -->
<a href="...">...</a>

<!-- в новом окне -->
<a href="..." target="_blank">...</a>

<!-- переход на заголовок внутри страницы -->
<h1 id="zagolovolovok">...</h2>
<a href="#zagolovolovok">...</a>

<!-- ссылка на email -->
<a href="mailto:zulutime@mail.ru">...</a>

<!-- ссылка с всплывающей при наведении подсказкой -->
<a href="..." title="podskazka">...</a>
```
